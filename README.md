# Web Push Notifications demo

## Requirements

* Firefox 44+

## Running demo

1. Install dependences:

    ```
    npm install
    ```

2. Build javascript:

    ```
    tsc
    ```

3. Run the server:

    ```
    npm start
    ```

4. Open the app at https://127.0.0.1:7000/index.html (Note: You will need to add a security exception)
